# QuirozTech - Landing Page de Soporte TI

Este repositorio contiene el código fuente de la landing page para **QuirozTech**, un servicio profesional de soporte técnico ubicado en **Quilpué, Chile**. El sitio está diseñado para ser ligero, eficiente y facilitar el contacto directo con los clientes.

## 🛠️ Servicios Ofrecidos

La plataforma detalla los servicios clave de soporte técnico:

- **Formateo e instalación**: Configuración profesional de Windows y Linux.
- **Eliminación de virus**: Limpieza completa de malware.
- **Optimización**: Mejora del rendimiento del PC.
- **Limpieza de hardware**: Mantenimiento preventivo y limpieza de componentes internos.
- **Recuperación de datos**: Rescate de archivos en discos dañados o eliminados.

## 🚀 Tecnologías Utilizadas

El proyecto fue construido utilizando estándares modernos de desarrollo web para asegurar rapidez y compatibilidad:

- **HTML5**: Estructura semántica del sitio.
- **CSS3**: Diseño modular dividido en `base.css`, `layout.css` y `components.css`.
- **Diseño Responsivo**: Adaptado para dispositivos móviles y escritorio.

## 📂 Estructura del Proyecto

```text
QuirozTech-landing/
├── css/
│   ├── base.css       # Estilos base y reseteo
│   ├── layout.css     # Estructura y contenedores
│   └── components.css # Estilos de tarjetas, botones y navegación
├── assets/            # Imágenes y recursos (Logo, iconos)
└── index.html         # Archivo principal de la landing page
```
